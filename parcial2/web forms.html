ASP.NET WEB FORMS ES
  • Se basa en la tecnología de Microsoft ASP.NET
  • Es compatible con cualquier explorador o dispositivo móvil
  • Se basa en Microsoft .NET Framework.
  • Es flexible porque puede agregar en ellos controles creados por el usuario y de terceros.
ASP.NET WEB FORMS OFRECE
  • Separación de HTML y otro código de interfaz de usuario
  • Amplio conjunto de controles de servidor
  • Enlace de datos eficaz, con gran compatibilidad con herramientas.
  • Compatibilidad con scripting
VENTAJAS DE ASP.NET WEB
  • Controles de servido
  • Trabajar con datos
  • Script de cliente y marcos de cliente
  • Administración de estados
VENTAJAS BASADA EN WEB FORMS
  • Admite un modelo de eventos que conserva el estado sobre HTTP
  • facilitar la administración de la información de estado.
  • Funciona bien para equipos pequeños de desarrolladores web y diseñadores
