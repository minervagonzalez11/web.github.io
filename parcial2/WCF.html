PASOS PARA LA CREACIÓN Y CONSUMO DE UN WCF

  1. Abrir la aplicación de visual studio
  2. Crear un nuevo proyecto
  3. Buscar la que dice aplicación de servicios wcf
  4. Cambiar el nombre del proyecto
  5. cambiar interface IService1 por Ipersona y al archivo
  6. Cambiar service1 por personaservice y al archivo
  7. crear base de datos
  8. meter datos
  9. crear carpeta de modelo
  10. adentro de esa agregar la base de datos
  11. en Ipersona definer el método consulta
  12. en personaservice utilizar entity framework
  13. crear proyecto web forms en aplicación web asp.net
  14. llamarlo cliente
  15. añadir una referencia
  16. agregar formulario web forms y llamarlo pagina
  17. agregar una grilla donde se mostrara todos los datos
